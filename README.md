# CLI_ToDo

A simple command line to-do list application.

## Usage

```bash
# Add a new task
python3 main.py add "Buy milk"

# List tasks
python3 main.py list

# Update the first task (number is optional, defaults to 1)
python3 main.py update 1 "Buy almond milk"
# or without specifying the number
python3 main.py update "Buy almond milk"
```
